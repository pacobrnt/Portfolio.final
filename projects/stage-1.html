À propos du projet

Cooking Recipe API est une plateforme permettant aux développeurs d'accéder à des recettes de cuisine via une API REST. Les utilisateurs s'inscrivent, obtiennent une clé API et suivent leur consommation depuis un tableau de bord.

Ma contribution : développement des vues, création des recettes et intégration d'un agent OpenAI pour générer automatiquement le contenu avec des données standardisées.
